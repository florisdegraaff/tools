# Modules

This folder contains one description file per module in the project.

## Suggested Convention

- File name: `<module-name>.md`
- One file per module
- Keep each file concise and practical

## Suggested Template

```md
# <Module Name>

## Purpose
What this module is for.

## Features
- Feature 1
- Feature 2

## Inputs
- Input A

## Outputs
- Output A

## Notes
Any important setup, constraints, or follow-up ideas.
```
